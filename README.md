# portspectre-dist
PORTSPECTRE distribution channel — compiled release binaries only (no source). Update feed for the in-app self-updater.

**Not a port scanner.** PORTSPECTRE is a *passive* port **monitor** / network-defense tray app for Windows: it watches your own machine's connections and raises alerts. It never scans or probes remote hosts. If you arrived here looking for an unrelated similarly-named scanning tool, this isn't it.
